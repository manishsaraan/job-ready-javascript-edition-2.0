1.) Naming Branches : 

"bug/fixed-all-caps"
"feature/giant-duck-modal"
"refactor/add-prop-types"
"style/everything-is-black"

2.) Commit with Commit Body
git commit -m "Short Message" -m "Long description"

3.)Reverting a commit
git revert commitHash

4.)Deleting commits with rebase
git rebase -i HEAD~2(latest two commits)

5.)Deleting the commit but not the code
git reset commitHash
